# Especificación de Requerimientos de Software (SRS)
Versión 1.1

## 1. Introducción
Este documento define los requerimientos para el sistema de tráfico inteligente (SmartTraffic). Se han añadido nuevas especificaciones para seguridad.

## 2. Descripción General
El sistema gestiona semáforos inteligentes mediante visión por computadora y protocolos de baja latencia.

## 3. Requerimientos Funcionales
3.1 Detección multi-modal (Cámaras y LiDAR).
3.2 Optimización de flujo basada en IA.
3.3 Interfaz para autoridades de tránsito.

## 4. Seguridad (NUEVA SECCIÓN)
El sistema debe estar encriptado de extremo a extremo para evitar ciberataques.

## 5. Glosario
OTX: Optimal Transport eXtreme para matching de flujos.
SmartTraffic: Nombre comercial del sistema.

# Especificación de Requerimientos de Software (SRS)
Versión 1.0

## 1. Introducción
El propósito de este documento es definir los requerimientos funcionales para el sistema de control de tráfico.

## 2. Descripción General
El sistema debe gestionar las luces de tráfico en tiempo real basándose en la densidad de vehículos.

## 3. Requerimientos Funcionales
3.1 El sistema debe detectar vehículos usando cámaras.
3.2 El sistema debe ajustar los tiempos de los semáforos óptimamente.

## 4. Requerimientos No Funcionales
El sistema debe tener una disponibilidad del 99.9%.

## 5. Glosario
OTX: Optimal Transport eXtreme.
